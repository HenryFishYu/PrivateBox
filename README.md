# PrivateBox

It's a PrivateBox(Like DropBox),the final goal is play videos online with html5 player<br />
一个私人云网盘，最终目标是实现上传视频的在线播放（H5）<br />
Attention:<br />
if you want to use it,edit src/main/resources/application.yml change the mysql config and insert ROLE_USER into table role.<br />
Then run Run.java with Spring-Boot Appilication.<br />
如果要运行它,先编辑src/main/resources/application.yml，然后修改数据库配置并在role表中插入一条数据:ROLE_USER。运行方式:<br />
以Spring-Boot的方式启动Run.java<br />
